- 入力を全部消す機能を実装しました
- 一文字消去を追加しました。スライス機能を使いgetで得た物の末尾を消し、表示をデリートし末尾が消えたものをインサートして実装しました

- 全消去と一文字消去は分かりやすいように色を変えました
